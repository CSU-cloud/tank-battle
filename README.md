# 🎮 坦克大战 (Tank Battle)

经典的HTML5坦克大战游戏，灵感来自经典FC游戏《Battle City》。

## 🕹️ 游戏特性

- 🎯 经典坦克大战玩法
- 🗺️ 随机生成的地图关卡
- 💥 丰富的爆炸特效
- 🛡️ 玩家护盾系统
- 🏆 得分系统
- 📈 多关卡递增难度
- 🧱 多种地形：砖墙、钢墙、水域、树林

## 🎮 操作说明

| 按键 | 功能 |
|------|------|
| ↑ ↓ ← → 或 W A S D | 移动坦克 |
| 空格键 | 发射炮弹 |
| P | 暂停游戏 |

## 🗺️ 地形说明

| 地形 | 说明 |
|------|------|
| 🧱 砖墙 | 可以被子弹摧毁 |
| 🔩 钢墙 | 无法被子弹摧毁 |
| 🌊 水域 | 坦克无法通过 |
| 🌲 树林 | 可以隐藏坦克 |
| 🏁 基地 | 需要保护的目标 |

## 🚀 如何运行

1. 克隆仓库
2. 用浏览器打开 `index.html` 文件即可开始游戏

或者直接访问 GitHub Pages 在线游玩。

## 📸 游戏截图

打开 `index.html` 后即可体验精彩游戏！

## 🛠️ 技术栈

- 纯 HTML5 + CSS3 + JavaScript
- Canvas 2D 渲染
- 无需任何第三方库
